# PRD: 전문가 관리 기능 (Expert Management)

## 1. 개요
- **목표:** 경제/주식 전문가들의 트위터 계정을 시스템에 등록, 수정, 삭제할 수 있는 관리 기능을 제공한다.
- **주요 작업:** 전문가 정보 CRUD API 구현, 관리자 전용 UI 개발.

## 2. 사용자 스토리
- "관리자는 새로운 전문가의 이름과 트위터 핸들을 입력하여 등록할 수 있다."
- "관리자는 등록된 전문가 리스트를 한눈에 확인하고, 더 이상 필요 없는 계정은 삭제할 수 있다."
- "전문가 정보가 추가되면, 다음 파이프라인 실행 시 해당 계정의 피드가 자동으로 수집되어야 한다."

## 3. 기술적 상세 및 요구사항

### 3.1 데이터 구조 (`ts_experts`)
- **name:** 전문가 표시 이름 (예: 김주식)
- **twitter_handle:** 트위터 아이디 (예: stock_king)
- **is_active:** 수집 활성화 여부 (기본값: true)
- **last_synced_at:** 마지막 피드 동기화 시점 (수집 로직에서 업데이트)

### 3.2 주요 기능
1.  **목록 조회:** DB에 등록된 모든 전문가 정보를 테이블 형태로 표시.
2.  **신규 등록:** 이름과 핸들을 입력받아 유효성 검사(Zod) 후 저장.
3.  **삭제:** 전문가 삭제 시 연관된 피드 데이터 처리 정책 결정 (현재는 단순 삭제).
