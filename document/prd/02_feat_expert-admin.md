# PRD: 어드민 관리 기능 (Admin Management: Experts & Stocks)

## 1. 개요
- **목표:** 전문가 프로필과 종목 마스터 데이터를 시스템에 등록, 수정, 검증할 수 있는 통합 관리 기능을 제공한다.
- **UI 구조:** shadcn/ui Sidebar를 기반으로 한 좌측 내비게이션 구조를 채택하고, 각 기능을 독립된 페이지로 분리 완료.

---

## 2. UI/UX 구조 (Sidebar Navigation)

### 2.1 사이드바 메뉴 구성
- **Dashboard:** 시스템 전반의 요약 정보 (통계 카드 및 시스템 현황)
- **전문가 관리 (Experts):** 전문가 리스트 및 등록/삭제 (`/admin/experts`)
- **종목 마스터 (Stocks):** 종목 정규화 및 검증 관리 (`/admin/stocks`)
- **시스템 로그 (Logs):** 파이프라인 실행 이력 확인 (`/admin/logs`)

### 2.2 레이아웃 설계 (`/admin/layout.tsx`)
- 모든 어드민 하위 페이지는 사이드바를 공유하며, `SidebarProvider`를 통해 상태 관리.
- `TooltipProvider`를 내장하여 사이드바 아이콘 호버 시 툴팁 제공.

---

## 3. 기능별 상세 요구사항

### 3.1 전문가 관리 (`/admin/experts`)
- 전문가 리스트 조회 및 신규 전문가(Twitter Handle 기반) 등록 기능.
- 전문가 삭제 시 연관 데이터 처리.

### 3.2 종목 마스터 관리 (`/admin/stocks`)
- **종목 정규화:** AI가 추출한 임시 티커(예: `UNKN_...`)를 공식 티커(예: `SpaceX`)로 수정 가능.
- **승인 시스템:** `is_verified` 토글을 통해 검증된 종목만 메인 서비스에 노출하도록 제어.
- **시장 분류:** KR, US, Global 라디오 그룹을 통한 명확한 분류 설정.
- **지능형 병합 (Merge):** 
    - 별칭(Aliases) 입력을 통해 오타나 다른 명칭을 공식 종목으로 통합.
    - 병합 시 기존 인사이트의 티커를 소급 적용하여 일괄 업데이트.
- **카운트 동기화:** `refresh_all_stock_counts` RPC를 호출하여 실제 언급 횟수 전수 재계산.
