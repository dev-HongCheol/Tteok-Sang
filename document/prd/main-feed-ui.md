# PRD: 메인 피드 UI 및 통합 필터링 시스템 (Main Feed UI)

## 1. 개요
- **목표:** 전문가로부터 수집 및 분석된 경제 인사이트를 사용자가 한눈에 확인하고, 테마/계정/기간별로 정밀하게 필터링하여 탐색할 수 있는 고도화된 메인 화면을 구축한다.
- **주요 작업:** 다중 선택 필터 시스템 구현, 인사이트 카드 UI 개발, 전역 다크모드 적용, 실시간 데이터 연동.

## 2. 사용자 스토리
- "사용자는 여러 개의 경제 테마와 여러 명의 분석 계정을 동시에 선택하여 맞춤형 피드를 구성할 수 있어야 한다."
- "사용자는 기간 선택기를 통해 과거의 특정 시점의 경제 인사이트를 조회할 수 있어야 한다."
- "사용자는 선택된 필터의 상태(예: 선택된 계정 수)를 항상 명확하게 인지할 수 있어야 한다."
- "사용자는 검색어를 통해 요약 내용이나 원본 트윗 본문 내의 키워드를 빠르게 찾을 수 있어야 한다."

## 3. 기술적 상세 및 요구사항

### 3.1 필터링 및 데이터 로직
- **통합 조회 API:** `expertIds`, `categories`, `importances`, `dateRange`, `searchQuery`를 동시에 처리하는 Supabase 조인 쿼리 구현.
- **필터 기본값:**
    - **중요도(Importance):** 초기 진입 시 `High` 등급만 선택되어 있음.
    - **기간:** 오늘 날짜가 기본 선택됨.
- **상태 관리 규칙:** 
    - `undefined`: 초기 상태 (해당 필터 무시, 전체 조회).
    - `[]` (빈 배열): 명시적 선택 해제 (결과 없음 처리).

### 3.2 디자인 및 UX 사양
- **테마:** 시스템 설정과 무관하게 **다크모드(Dark Mode) 고정** 적용.
- **인사이트 카드:**
    - 중요도(`High`, `Medium`, `Low`)에 따른 배지 색상 및 농도 차별화.
    - **원문 펼쳐보기:** AI 요약 아래에 원본 피드 본문(`raw_content`)을 배치하고, 기본 2줄 제한(`line-clamp`) 후 클릭 시 전체 노출.
    - **관련성 점수:** 0~100% 점수를 프로그레스 바 형태의 시각적 지표로 제공.
    - 전문가 아이디(@handle) 강조 및 원본 트윗 연결 링크 제공.
- **필터 UI:** 
    - **중요도 필터:** 최상단에 배치하여 가장 핵심적인 데이터만 우선적으로 볼 수 있도록 유도.
    - 분석 계정 목록에 `ScrollArea` 적용 (최대 높이 300px).
    - 선택 상태를 하단에 상시 노출 ("xx개 계정 선택됨").
    - 필터 초기화 버튼을 통한 원클릭 리셋 기능 (중요도는 `High`로 리셋).
